Встроенный Просмотрщик
 
     Для просмотра графических файлов в программе присутствует собственный просмотрщик. Возможет как оконный, так и полноэкранный режим (Слайд-Шоу).

    Предыдущая фотография
    Следующая фотография
    Оптимальный размер
    Реальный размер
    Слайд-Шоу
    Во весь экран
    Уменьшить
    Увеличить

    Контекстное меню содержит следующие пункты:

Следующий - Перейти к следующей фотографии
Предыдущий - Перейти к предыдущей фотографии
Запустить таймер\Остановить таймер - Начать\Прекратить автоматическое переключение на следующую фотографию (можно настраивать)
Увеличить - Увеличить фотографию
Уменьшить - Уменьшить фотографию
Реальный размер - Показать фотографию в реальном размере
Наилучший размер - Показать фотографию целиком
Выполнить - Открыть фотографию в приложение, которое предназначено для открытия файлов этого типа
Скопировать - Скопировать файл в буфер обмена
Найти - Открыть в проводнике Windows папку с фотографией
Сделать рисунком рабочего стола - Использовать данную фотографию в качестве рисунка для рабочего стола
Повернуть изображение - Поворачивает изображение в указанном направлении. При повороте изображения в БД просто меняется свойства поворота в БД, что позволяет избежать потери качества про повороте изображений форматов JPEG GIF
Изменить размер - Позволяет изменяет размер фотографии
На весь экран - Переключение на полноэкранными режим.
Слайд-Шоу - Показать текущие фотографии в режиме Слайд-Шоу.
Пункт в БД - Просмотр и редактирование информации об этой фотографии
Свойства - Просмотр свойств фотографий
Добавить в БД - Добавить в базу данных эту фотографию или все фотографии, находящиеся в папке с данной фотографией
Перейти в окно поиска - Открыть окно поиска и вывести его на первый план
Проводник - Открыть папку с фотографией с встроенном проводнике
Выход - Закрыть данное окно

Правой кнопкой мыши также можно вызвать контекстное меню из проводника Windows к этой фотографии, нажав правой кнопкой мыши и отпустив её более чем через секунду.
