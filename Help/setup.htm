Установка
 
После запуска файла Setup.exe появится окно установки.
Если вы согласны с условиями распространения программы, то установите галочку "Я согласен с этими аргументами".
При необходимости можете изменить "Конечную Папку Программы", "Папку с Файлами Базы" Данных.
Путь к "базе данных по умолчанию" содержит первую найденную базу данных, находящуюся в папке с программой. Если её не нужно устанавливать, но можно нажать на кнопку BackSpace, удалив текущий путь, либо же выбрать другую базу данных. Этот пункт установки доступен только если на компьютер уже  установлена IDAPI.
Если у вас доступен выпадающий список "Тип БД" то вы можете выбрать какую базу данных использовать по умолчанию - Paradox или Access. Paradox использовать не рекомендуется и эта возможность оставлена в целях совместимости.
При необходимости нажмите установите галочку "Перемещать Личные Записи"
Нажмите на кнопку "Следующий"
Введите ваше имя
При необходимости отредактируйте список файлов, которые будут ассоциироваться с программой
Нажмите на кнопку "Установить"

После нажатия кнопки "Установить" программа будет установлена на компьютер.
