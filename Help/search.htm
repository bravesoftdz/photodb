Поиск фотографий
 
Окно поиска включает в себя:

Панель результатов поиска
Панель задания параметров поиска (в том числе и расширенная)
Панель просмотра и редактирования свойств выделенных фотографий
Панель проводника по папкам

     Панель результатов поиска содержит все фотографии, которые соответствовали критерию поиска.

Главное контекстное меню содержит следующие пункты:

 Слайд-Шоу - Открыть данный файл во встроенном просмотрщике или же в другой программе просмотра, указанной в настройках.
Выделить всё - Выделить все текущие фотографии
Показать окно обновления - Открыть окно обновления
Справка:
  Справка - Справка по программе
  Активация программы - Ввод кода для активации программы
  О программе - Окно "О программе"
  Сайт программы - Открыть страничку программы
  Связаться с автором - Написать письмо автору программы
Управление БД - Открыть окно "Управление БД"
Настройки - Отобразить окно настроек программы
  Редактор - открытие  графического редактора
Управление группами - Открыть окно управления группами
Сохранить как таблицу - Сохранить результаты поиска в виде таблицы (т.е. как базу данных)
Загрузить результаты - Загрузить результаты из файла результатов (*.ids *.ith *.dbl)
Сохранить результаты - Сохранить результаты в файле результатов (*.ids *.ith *.dbl)
  *.ids - файлы хранят ID фотографий, рекомендуется для временного сохранения результатов поиска. *.ith - файлы хранят индетефикаторы фотографий (IDEx), *.ith не привязывается к конкретной БД в отличии *.ids. *.dbl - Список файлов, может содержать как файлы, находящиеся в БД, так и другие файлы, частично читается в окне поиска, полную расшифровка файла выдаётся в Палели для хранения изображений.
Новая панель - Создать новую панель для для хранения промежуточных результатов
Новое окно поиска - Создаёт новое окно для поиска фотографий
Проводник - Открыть встроенный проводник

     Для поиска фотографий существует панель задания параметров поиска. Поиск осуществляется по любому тексту, введённому в окно. Ищется совпадение в имени файла, ключевых словах и комментариях. Если нужно найти фотографию по нескольким словам, то слова отделяются пробелами. При необходимости исключить из поиска фотографии с каким-либо словом, то перед ним ставится знак "-" (без кавычек). Пример: "Вася -Аня" (без кавычек) - поиск фотографий, у которых в имени файла либо в комментарии либо в ключевых словах содержится слово "Вася" и не содержится слово "Аня".
  Возможно выбирать поля, по которым будет производится поиск. Для этого нужно в строке поиска дописать команду [+коды_полей], в качестве полей доступно использовать имя файла (F), комментарий (C), ключевые слова (K), группы* (G) ссылки* (L). К примеру, если нужно найти фотографии, у которых в ключевых словах или в ссылках будет строка "Дом" и не будет строки "Дача", нужно написать "Дом -Дача [+KL]"
 * - т.к. данные поля имеют сложный формат, при поиске могут попадаться результаты, внешне не подходящие под запрос
     Возможны также следующие специальные команды поиска:

  :group(Имя_Группы):
поиск фотографий, относящихся к группе с именем [Имя_Группы]. При поиске учитывается указанный рейтинг и опция показа личных фотографий.

 :folder(Имя_Папки):
поиск фотографий, находящихся в папке с именем [Имя_Папки]. При поиске учитывается указанный рейтинг и опция показа личных фотографий.

  :folder(ID_Фоторгафии):
поиск фотографий, находящихся в папке, в которой находится фотография с ID, равным [ID_Фоторгафии]. При поиске учитывается указанный рейтинг и опция показа личных фотографий.

  ID_Фоторгафии1$ID_Фоторгафии2$ID_Фоторгафии3$...
поиск фотографий с ID, равным [ID_Фоторгафии1], [ID_Фоторгафии2], [ID_Фоторгафии3]... После каждого номера ID ставится знак $. При поиске учитывается указанный рейтинг и опция показа личных фотографий.

  :Dublicates:
поиск фотографий, отмеченных как дубликаты.

  :DeletedFiles:
поиск фотографий, отмеченных как удалённые, т.е. запись в базе данных присутствует о файле, но самого файла в указанном месте нет.

  :ThFile(Путь_К_Файлу):
поиск фотографий, информация о котором записана в файле с именем [Путь_К_Файлу].

  :similar(ID_фотографии):
поиск фотографий, которые идентичны данной (дубликаты)

  :Text(мой текст):
поиск фотографий, содержащих заданный текст. Поиск по полям комментария, ключевых слов и пути файла

  :KeyWord(ключевое слово):
поиск фотографий, содержащих данное ключевое слово (или часть)

  :Links:
поиск фотографий, содержащих хотя бы одну ссылку

  :ScanImageW(путь_к_изображению:коэффициент_похожести):
поиск фотографий, "похожих" на данное изображение. Возможно найти изображение в БД, даже если оно было частично преобразовано (перевод в JPEG или другой формат с потерей качества), перекодировано JPEG, даже если было повёрнуто на угол, кратный 90* (или близко к этому). При поиске учитываются все условия. Функция может быть вызвана из меню к изображению, если оно вызвано с нажатой клавишей Shift. Также находит изображения с похожей цветовой гаммой. (коэффициент_похожести может быть опущен, в этом случае он считается равным 20).

  :ScanImage(путь_к_изображению:коэффициент_похожести):
"облегчённый" аналог ScanImageW, более быстрый вариант, но менее качественный и не учитывает поворот изображения, не рекомендуется.

символ "%" перед специальным запросом сразу убирает все ограничения на поиск (если они возможны при запросе). Пример: "%:group(Друзья):" - поиск по группе "Друзья" по всей БД без ограничений.

     При выделении одной или нескольких фотографий появляется панель просмотра и редактирования свойств выделенных фотографий. Если необходимо изменить информацию о фотографии, то после изменения в панели свойств выделенных фотографий нужно нажать на кнопку "Сохранить", после чего будет произведено сохранение информации в базе данных. При выделении нескольких фотографий свойства "оценка", "дата" отображают наиболее часто встречающиеся данные свойства, "группы" и "ключевые слова" - только общие для всех выделенных объектов свойства.

     Для сортировки изображений нажмите мышкой по значку и выберите нужные режимы сортировки

     Для быстрого поиска фотографий какой-нибудь группы можно использовать кнопку , при нажатии на неё выводится список существующих в БД групп и по нажатию на любую из них производится поиск нужной группы

     Панель проводника по папкам служит для обзора фотографий в базе данных, принадлежаих конкретным папкам на диске. Данная панель открывается в контекстном меню над панелью задания условий поиска (обычной, не расширенной), где необходимо выбрать пункт пункт Панели->Проводник.

Правой кнопкой мыши также можно вызвать контекстное меню из проводника Windows к этой фотографии, нажав правой кнопкой мыши и отпустив её более чем через секунду.
