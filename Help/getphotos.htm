Получение фотографий
 
Диалог используется для быстрого получения фотографий со съёмных носителей.

Дата фотографии - если фотоаппарат записывает дату снимка в фотографии, то программа попытается определить дату, иначе вы можете сами изменить дату снимков

Маска имени папки - используется для определение имени конечной папки. В маске можно использовать следующие замены (в скобках дан пример для 8 апреля 2005 г.):

%yy:mm:dd (05.04.08)
%YMD (вторник, 8 апреля 2005 г.)
%coment - комментарий
'%yyyy  (2005)
%yy  (05)
%mmmdd (08 апреля)
%mmmd (8 апреля)
%mmm (апр)
%mm (08)
%m (8)
%dddd (пятница)
%ddd (Пт)
%dd (08)
%d (8)

Комментарий к папке - изменяет значение переменной %coment

Имя папки - результат действия маски

Конечная папка - папка, в которой будет создана новая папка

Получать мультимедийные файлы - указывает, что будут также получаться файлы фидео\аудио форматов (маска для файлов может задаваться вручную)
