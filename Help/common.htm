Общие сведения
 
    Программа рассчитана на работу в операционной системе Windows XP, но может нормально функционировать в среде Windows 2000*. Для работы достаточно CPU 600 Mg, 256 Mb оперативной памяти + 16 Мб видеопамяти, но это для работы с минимальными параметрами. Если нужна быстрая работа, плюс нормальная работа с достаточно большими фотографиями, желательно CPU от 1000 Mg + 512 MbRAM.Программа занимает около 4 мегабайт при установке (не учитывая файлов баз данных).

    Программа работает со следующими типами графических файлов:

TIFF images (*.tif; *.tiff)
GFI fax images (*.fax)
SGI images (*.bw, *.rgb, *.rgba, *.sgi)
Autodesk images files (*.cel; *.pic)
Truevision images (*.tga; *.vst; *.icb; *.vda; *.win)
ZSoft Paintbrush images (*.pcx, *.pcc)
Word 5.x screen capture files (*.scr)
Kodak Photo-CD images (*.pcd)
Portable pixel/gray map images (*.ppm, *.pgm, *.pbm)
Dr. Halo images (*.cut, *.pal)
CompuServe images (*.gif)
SGI Wavefront images (*.rla, *.rpf)
Standard Windows bitmap images (*.bmp, *.rle, *.dib)
JPEG Images (*.jpeg, *,jpg)
Photoshop images (*.psd, *.pdd)
Paintshop Pro images (*.psp)
Portable network graphic images (*.png)
RAW форматы многих камер

 

Дополнительная информация может находиться на официальной страничке программы:

http://www.illusdolphin.narod.ru/photodb

* - в данной операционной системе неверно отображаются иконки программы, рассчитанные на работу в Windows XP/Vista
