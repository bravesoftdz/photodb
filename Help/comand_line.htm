Командная строка

Программа может вызываться с параметрами командной строки для автоматизации некоотрых действий. Параметры не чувствительны к регистру. Вот полный перечень параметров с их описанием:

/OPTIMIZE_DUBLICTES - запускает при запуске программы оптимизацию дубликатов в текущей базе.

/SHOWBADLINKS - проверяет сканирование на несуществующие ссылки в текущей базе.

/RECREATETHTABLE - пересоздаёт предпросмотры изображений в базе, а также исправляет неточности описаний файлов в базе.

/BACKUP - запускает принудительно резервное копирование текущей базы.

/PACKTABLE - запускает упаковку базы.

/CONVERT - запускает конвертирование базы.

/SLEEP - после запуска ждёт одну секунду

/SAFEMODE - инициализирует "Безопасный режим", в котором отключаются некоторые функции.

/UNINSTALL - запускает диалог удаление программы

/EXPLORER - запускает проводник и папкой, указанной после параметра. К примеру: /explorer "c:\photos"

/GETPHOTOS - запускает получения фотографий с указанной папки. К примеру: /getphotos "c:\photos"

/NOLOGO - отключает окно "о программе" при старте.

/user user - указывает имя пользователя для автоматического входа

/pass pass - указывает пароль для автоматического входа (необходимо указания также пользователя)

/NoPrevVersion - отключает проверку запущенных копий программы

/NoFaultCheck - отключает проверку на крах программы

/NoFullRun - не запускает ни одно окно программы! После выполнения всех запросов програма выгрузится

/Execute - запускает скрипт БД. К примеру /Execute "c:\script.dbscript"

/NoVistaMsg - отключает окна сообщений Vista-стиля (если у на компьютере устновлена Vista)

/NoVistaFileDlg - отключает диалоги выбора файлов и папок Vista-стиля (если у на компьютере устновлена Vista)

/SelectDB "DBFile" - выбирается база из файла DBFile

/SelectDB "DBName" - выбирается база с внутренним именем DBName

/SelectDBPermanent - выбранная база устанавливается по умолчанию

/ADD "file" - добавляет файл

/ADD "directory" - добавляет директорию

/AddPass "pass1!pass2!..." - добавляет пароль или списов паролей. Пароли разделяются восклицательным знаком.

/SQLExec - выполняет SQL запрос на базу. Пример: /SQLExec "delete from imagetable where id=0". Не рекомендуется использовать данную комадну т.к. она является низкоуровневой и не содержит никаких проверок на выполняемые действия.

/SQLExecFile - выполняет SQL запрос на базу из файла. Пример: /SQLExec "c:\1.sql". Не рекомендуется использовать данную комадну т.к. она является низкоуровневой и не содержит никаких проверок на выполняемые действия.
