Работа с группами
Работа с группами содержит следующие пункты:

Управление существующими группами
Создание новой группы
Редактирование существующей группы
Изменение списка групп для фотографии\фотографий

     Для создания новой группы необходимо:
 В окне управления группами выбрать в контекстном меню для любой группы пункт "Добавление группы"
В редакторе групп нажать на кнопку "Новая группа"
В открывшемся окне задать имя новой группы, комментарий, изображение, после чего нажать на кнопку "Да"

     Менеджер групп предназначен для отображения и редактирования групп, находящихся в базе данных. Для открытия менеджера нужно выбрать пункт "Управление группами" в меню из окна поиска или из меню окна для просмотра группы (пункт "Управление") или же из редактора списка групп по нажатию на кнопку "Управление". Если в базе данных на данный момент нет ни одной группы, то будет открыто окно создания новой группы. Меню к каждой группе (вызывается правым кликом по изображению группы) содержит следующие пункты:

Изменение группы - Изменение текущей группы
Удаление группы - Удаление текущей группы
Добавление группы - Добавление новой группы
Изменение прав - Отображает диалог изменения прав к текущему пользователю
Поиск фотографий группы - Открыть окно поиска и отобразить в нём все фотографии данной группы
Информация по группе - Отобразить краткую сводку о выделенной группе

     При редактировании группы необходимо в окне изменения группы изменить название группы (оно не должно совпадать с уже имеющимися группами), изменить при необходимости изображение группы и комментарий группы, после чего нажать на кнопку "Да"

     Для изменения списка групп для фотографии\фотографий существует окно "Редактирование групп". В списке "Текущие группы" содержатся текущие группы. Чтобы удалить группу из списка нужно выбрать в контекстном меню пункт "Удалить" или нажать на кнопку Delete. Для добавления группы нужно выбрать новую группу в списке "Доступные группы", после чего эта группа отобразится в списке. Кнопка "Управление" вызывает менеджер групп. "Новая группа" - создание новой группы. По нажатию ан кнопку "Да" изменения вступят в силу.

Вся информация о группах для Paradox-баз данных содержится в файлах с именем "ИмяОсновнойТаблицыG.MB" и "ИмяОсновнойТаблицыG.DB", лежащих в папке с основными файлами к базе данных, для Access-баз все данные находятся в одном основном файле.
