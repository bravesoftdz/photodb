Скрипты
 

ВВЕДЕНИЕ

 

Вот краткое описание языка скриптов, используемом в БД. Если кт-то захочет узнать что-либо специфическое, то спрашивайте по e-mail т.к. не думаю, что большое количество людей имеют склонности к программированию (а тем более желание) на незнакомом языке.

Язык синтаксисом похож в некотором на PHP, в некотором на JavaScript, но возможностей естественно меньше. Регистрозависим!

Переменные начинаются с символа “$”

Символ присваивания “=”

Оператор заканчивается точкой с запятой “;”

 

$var =  значение;

 

Существует только 8 типов:

 

INTEGER – целое

STRING – строка (ограничивается двойными кавычками “строка”)

BOOL – булеан (true или false)

FLAOT – с плавающей точкой

ARINTEGER – массив целых

ARSTRING – массив строк

ARBOOL – массив булеанов

ARFLOAT – массив с плавающей точкой

 

$a = 10;

$b = $a;

$s = “string”;

$n = 3.45;

$b = true;

 

Работа с массивами не очень удобная, но всё же. Массивы начинаются с нуля.

 

$ar = GetIntArray(10);

SetIntItem($ar,5,20); //устанавливает 6му элементу значение 20;

AddIntItem($ar,4); //увеличивает размер массива на 1 и устанавливает значение последнего элемента = 4

$int=GetIntItem($ar,5); //устанавливает $int=20;

 

А также GetStringArray(int), GetBoolArray(int), GetFloatArray(int)

 

Для целых чисел возможны вычисления по схеме:

$b=5;

$int=%34+cos(45)*$b;

Значение округляется до целого.

 

 

УСЛОВИЯ

 

Присутствует if.

if (условие)

{

            если_верно;

}; else

{

            если_ложно;

}; // обязательно наличие в таких конструкциях точки с запятой!!!

 

условие пока не очень хорошие. Если в начале присутствует # то вычисляется выражение в скобках и если оно НЕ НОЛЬ то считается верным, иначе – ложным.

 

if (#($b<10)&($b>0)) {}; //выполнится в скобках если $b от 1го до 9ти.

 

& - логическое И

| - логическое ИЛИ

~ - логическое ИЛИ-НЕ

простого НЕ нету. Реализуется при помощи ИЛИ-НЕ

(true)~1 -> false

                    

Либо можно указывать простое условие (одно сравнение)

 

$b=true;

if ($b=true) {};

if ($i<>20) {}; else {};

 

Возможно использование > < = <>

 

 

ЦИКЛЫ

 

Существует только один тип for (пока), но его вполне хватает.

for(инициальизация; условие; выполняется_для_каждого)

{

            тело_цикла;

};

 

for ($i=0;$i<10;$i=%$i+1)

{

            тело_цикла;

};

@functions

.int                               целое

.float[]                         массив значений с плавающей точкой

МЕНЮ

 

Многие меню в БД созданы при помощи скриптов. Меню задаётся схемой

 

<надпись>[номер_иконки]

{

  initialization:

   выполнение_секции_инициализации;

  run:

   выполняется_по_клику_меню;

}

 

Если секции не указаны то код будет выполняться по клику на меню.

 

<#>[]{путь_к_скрипту} //включат в файл содержимое другого скрипта

<>[]{путь_к_скрипту} //включат в файл содержимое части меню

<->[]{} // разделитель

 

Меню создаются или динамически или статически. Для динамического создания используются функции CreateItem, CreateItemDef и CreateItemDefChecked.

 

InVisible;  //устанавливает флаг Visible в false – меню невидимо

 

Флаги меню, которые можно изменить в секции инициализации

$Tag  //целое число – индетефикатор меню

$Default //меню выделено

$Enabled //доступно

$Checked //установлен флаг

$Visible //видимо

$Caption //заголовок
