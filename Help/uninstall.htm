Удаление
 

Для удаления программы с компьютера нажмите:

Пуск->Панель Управления->Установка и Удаление Программ

Выберите "Photo DataBase" и нажмите кнопку "Изменить\удалить" (Change\Remove). На вопрос "Вы действительно хотите удалить программу?" ответьте ДА.

Выберите необходимые компоненты для удаления

Нажмите на кнопку "Удаление"

После нажатия на кнопку "Удаление", программные компоненты будут удалены с компьютера.

Перед удалением программы убедитесь, что никакие ваши фотографии в данный момент не защищены паролем с помощью данной программы! Иначе никакие другие программы не позволят вам открыть эти файлы!
