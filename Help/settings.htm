Настройки
 
Закладка "Общие":

Цветовая схема - текущие цвета, используемые программой.
Загрузить тему - Применить изменния цветевой гаммы
Сброс - Загрузить текущие программные цвета в это окно
Загрузить из файла - Загрузить цветовую схему из файла (*.dbc)
Сохранить текущую тему в файл - Сохранить цветовую схему в файл (*.dbc)

Файлы к темами для программы хранятся в папке Themes в папке программы.

Показывать предпросмотр - определяет, будет ли появляться при наведении курсора на изображение с окне поиска или проводнике уменьшенный предпросмотр изображения.
 Анимированный показ - Окно предпросмотра будет плавно появляться и исчезать
 Показ тени - Определяет, будет ли показываться тень под окном предпросмотра.

Кнопка Пользователи вызывает менеджер пользователей.

Доступные цветовые схемы - отображает список тем для быстрой загрузки. При выделении нужной темы и нажатию на кнопку "Загрузить тему", тема будет загружена в программу.

Закладка "Проводник":

Показывать текущие объекты - определяет, какие объекты будут отображаться во встроенном проводнике.
Папки - будут отображаться папки
Простые файлы - будут отображаться все файлы, не поддерживаемые программой
Изображения (изображениями считаются все типы файлов, поддерживаемые программой)
Скрытые файлы - будут отображаться скрытые файлы

Опции предпросмотров - определяет показ предпросмотр объектов в проводнике.
Показывать атрибуты - Определяет, будут ли выводится вместе с предпросмотром оценка фотографии, установленный поворот и доступ (личная или общая).
Показывать предпросмотр для папок - Будет генерироваться предпросмотр к папкам на основе первых четырёх изображений, найдённых в ней.
Сохранять предпросмотр для папок - определяет кэширование предпросмотра для более быстро повторного отображения
Показывать предпросмотр для изображений - Будет отображаться предпросмотр к изображениям.

Отображать EXIF заголовок - помечает фотографии с EXIF  заголовком иконкой
Отображать ссылки "Другие места" - показывает, будут ли отображаться ссылки "Другие места" ("Мои Документы", "Мои Рисунки", "Мой Компьютер", "Рабочий стол")
Дополнительные места - Возможно задать дополнительные ссылки в окне проводника для удобства навигации. "Отображать в" - показывает, где будет отображаться данная ссылка.

Текущие расширения - расширения, доступные для приложения. По кнопке "Установить" изменения вступают в силу. Для изменения списка используемых расширений нужно правой кнопкой мыши выбрать соответствующее действие для любого типа файлов из списка.

Очистить кэш предпросмотров - очищает все изображения директорий, сохранённые программой в памяти для ускорения вторичной загрузки директории.

Очистить кэш иконок - очищает все иконки для файлов, сохранённые программой в памяти для ускорения вторичной загрузки директории.

Закладка "Просмотр":

     Слайд-Шоу - определяет настройки просмотра фотографий.
Количество шагов для Слайд-Шоу - для режима Слайд-Шоу определяет за сколько шагов будет осуществлён переход от одного изображения к другому
Дополнительная задержка для Слайд-Шоу - для режима Слайд-Шоу определяет дополнительную задержку между кадрами.
Скорость перехода в чёрно-белый цвет - для оконного режима, определяет скорость перехода в чёрно-белый режим при загрузке изображения, слишком маленькая скорость (а соответственно и большое количество шагов может заметно увеличить время загрузки изображения)
Скорость слайдов для полноэкранного режима  - для полноэкранного режима определяет количество миллисекунд между кадрами
Поворачивать изображение на диске не спрашивая подтверждения - в окне просмотра при выборе пункта поворота изображения не в БД изображение будет поворачиваться без вызова окна настроек
Даже если файл в БД, поворачивать на диске - В любом случае файл будет поворачиваться на диске без задания опций поворота в БД (удобно если после будет использоваться внешний вьювер)
Опции JPEG - Если будет использоваться поворот изображения из окна просмотра, то будут использоваться данные настройки
Следующий при клике - переход к следующему кадру при клике мыши в окне.
Использовать качественную прорисовку - Использовать сглаживание при отображении изображений
Использовать другой просмотрщик - Использовать нестандартный просмотрщик. В окно ниже необходимо ввести путь к нему с указанием параметров. Вместо имени файла\файлов нужно поставить %S.

Закладка "Меню":

     Данные настройки позволяют добавлять свои пункты в меню программы для более тесной интеграции с системой. По умолчанию там есть два пункта, созданных для примера.

Пункт меню - Отображает все текущие пункты дополнительного меню

      "Заголовок", "Исполняемый файл", "Параметры запуска", "Иконка", "Добавить в субменю" - отдельные параметры для каждого пункта меню.
      Для добавления нового пункта меню достаточно нажать на кнопку "Добавить" и найти в проводнике исполняемый файл. Обычно стандартные параметры хорошо подходят для всех приложений. Далее достаточно нажать на кнопку "Да" чтобы изменения вступили в силу.
      "Добавить в субменю" - определяет, будет ли данный пункт внесён в субменю или он будет добавлен как отдельный пункт меню.

Закладка "Безопасность":

Автоматически сохранять пароли для текущей сессии - Автоматически устанавливает галочку "Сохранять пароль на время работы" в диалогах шифрования
Автоматически сохранять пароли в настройках - Автоматически устанавливает галочку "Сохранять пароль в настройках" в диалогах шифрования
Очистить текущие пароли в сессии - Очищает текущие загруженные пароли
Очистить текущие пароли в настройках - Очищает текущие пароли сохранённые в настройках.
Создавать резервную копию - частота создания резервных копий текущей БД

Закладка "Глобальные":

Использовать выделение при наведении в списках - определяет, будет ли выделятся пункты в списках при наведении мыши
Сортировать группы - будут ли сортированы группы в алфавитном порядке
Использовать GDI+ - использовать графическое ядро нового поколения (драйвера установлены на всех современных компьютерах и включены в Windows XP SP2.
Опции JPEG -  глобальные опии кодирования в JPEG
Разбивать на группы в поисковике -  в окне поиска будут ли разбиваться на группы результаты поиска (по критерию поиска)
Разрешить загрузку нескольких копий окна свойств - при установки опции свойства разных файлов будут отображаться в разных окнах свойств, иначе только в единственном окне.
Показывать главное меня в окне поиска - будет ли отображаться главное окно в окне поиска (для возможности восстановления "старого вида")
Виртуальный курсор в редакторе - установите данную опцию если возникают проблемы с рисованием кистью в редакторе при больших размерах курсора
Следить за изменением файлов и обновлять ссылки - при установленной опции при изменении изображения будет произведён в БД поиск ссылок на данное изображение и если они будут найдены - то ссылки будут скорректированы. Это полезно при использовании ссылок типа IDEx но немного замедляет работу.
Запускать проводник при запуске - определяет что по умолчанию вместо окна поиска будет открываться окно проводника или с какой-то определённой папкой (опция "Использовать папку") или при выключенной опции будет открыта последняя папка с предыдщей работы программы.
Не добавлять в БД файлы, размером меньше - при добавлении в БД будут игнорироваться файлы размером меньше указанного. полезно когда добавляется большая директория которая может содержать html-файлы с элементами дизайна.
