Основное меню свойств фотографии
 
     Для каждой фотографии доступны следующие пункты (если фотографии нет на диске, то недоступны пункты, относящиеся к файлу):

Показать - Открыть данный файл во встроенном просмотрщике или же в другой программе просмотра, указанной в настройках.
Выполнить - Открыть фотографию в приложение, которое предназначено для открытия файлов этого типа
Обновить - обновить информацию о данной фотографии из Базы Данных
Поворот (нет\90*\180*\270*) - Поворот фотографии (текущее значение выделено жирным шрифтом)
Оценка (0\1\2\3\4\5) - текущая оценка фотографии (текущее значение выделено жирным шрифтом)
Сделать личной\Сделать общей - Пометить фотографию как личную\общую
Дата - Просмотр и редактирование даты и времени к фотографии
Группы - Просмотр и редактирование групп, к которым относится данная фотография
Шифрование - установка пароля к фотографиям, а также их расшифровка и ввод пароля
Сделать рисунком рабочего стола - Устанавливает фотографию на рабочий стол (только для незашифрованных фотографий)
Обновить запись в БД - обновляет информацию о файле если он был видоизменён\отредактирован извне.
Дубликаты - меню доступно для файлов, помеченных программой как дубликаты. "Показать дубликаты" - отображает в новом окне поиска все дубликаты этой фотографии. "Удалить другие дубликаты" - удаляет все остальные дубликаты этой фотографии как из базы данных, так и физически!
Удалить информацию из БД - Удалить информацию о данной фотографии из БД
Удалить файл - Удалить данную фотография с диска
Копировать элемент - Копировать файл в буфер обмена
Переименовать - Переименовать файл
Проводник - Открыть папку с данным файлом в встроенном проводнике
Отправить в - Отправить папку в новую или существующую панель для временного хранения
Показать папку - Отобразить все фотографии в БД, которые относятся к папке, в которой лежит данная фотография
Свойства - Отобразить свойства фотографии

       Кроме того данном меню могут быть дополнительные пункты, которые можно создать в окне настроек программы.

При установлении поворота к фотографии, исходный файл не изменяется, а лишь изображается с соответствующем повороте, что избегает возможных потерь при перекодировании JPEG и даёт возможность поворачивать файлы "экзотических" форматов.

Удаление любой информации необратимо!
