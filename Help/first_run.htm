Первый запуск
 
     При первом запуске программы необходимо создать учётную запись администратора. Для этого в открывшемся окне нужно ввести пароль, подтверждение и при необходимости сменить изображение для этой учётной записи. После нажатии на кнопку "Да" запись будет создана.

     При первом запуске программы будет выведен диалог добавления файлов в БД. При помощи его вы можете указать папки которые будут добавлены в БД, при этом нет ограничений на количество изображений.

 Если на компьютере уже стояла данная программа более ранней версии, то рекомендуется обратится к данной странице.
