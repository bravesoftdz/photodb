Для создания нового пользователя нужно выбрать пункт "Добавить нового пользователя" в контекстном меню, которое вызывается при клике правой кнопки мыши по картинке любого пользователя. В появившемся окне выбрать имя для нового пользователя, его изображение и пароль, ввести подтверждение пароля и нажать на кнопку "Да". После этого будет создан новый пользователь.

Для управления пользователями существует окно управления пользователями, которое вызывается из окна свойств программы (кнопка "Пользователи" на закладке "Общие").

Для удаления пользователя необходим выбрать пункт "Удалить пользователя" в контекстном меню, которое вызывается при клике правой кнопки мыши по картинке соответствующего пользователя. Это действие нельзя отменить.

Для редактирование существующего пользователя или администратора необходимо выбрать пункт "Изменить пользователя" в контекстном меню, которое вызывается при клике правой кнопки мыши по картинке соответствующего пользователя. В появившемся окне можно изменить изображение пользователя, а также пароль. Необходимо ввести существующий пароль, новый пароль и его подтверждение. (для администратора существующий пароль вводить не нужно)

Для редактирование прав пользователей нужно войти в БД под Администратором. Тогда в контекстном меню в изображении пользователя нужно выбрать пункт "Изменить права" и нажать на кнопку "Да".

У каждого пользователя за исключением Администратора можно изменять права, определяющие возможности пользователя. При создании нового пользователя у него не задаётся никаких прав, поэтому нужно выбрать все необходимые права после создания учётной записи.

Для выбора изображения для существующего (нового) пользователя в окне изменения (создания) пользователя нужно выбрать в контекстном меню в изображении пользователя пункт "Загрузить из файла" и выбрать необходимое изображение, или же при клике левой кнопки мыши выбрать в появившемся окне необходимое изображения из списка (данные изображения входят в поставку программы в виде plug-in'ов и необязательны, хранятся в папке PlugIns, а при их отсутствии данное окно появляться не будет).

Если напротив пользователя стоит галочка "Авто вход", то по умолчанию будет загружен данный пользователь. Чтобы при запуске программы появилось окно входа (без автоматического входа) необходимо при запуске удерживать нлавишу ctlr или shift. Для установления режима автоматического входа в систему необходимо зайти в систему при установленной галочке "Авто вход". При этом пароль будет запомнен программой. Если вход будет осуществлён без галочки для пользователя, который загружался ранее автоматически, то этот режим будет отменён. Также его можно отменить из окна управления пользователями.

Изменение прав пользователей как и многопользовательская работа организована для частичной защиты данных в БД, но в связи с тем что пользователь всё равно может найти как саму БД на диске, так и файлы, для защиты лучше использовать шифрование фотографий и резервное копирование БД.
