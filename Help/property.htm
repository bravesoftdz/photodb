Добавление, просмотр и редактирование информации к фотографиям
     Для просмотра свойств фотографий из базы данных (как одной, так и нескольких), а также для отображения свойств других изображений существует окно свойств, вызываемое по нажатию на пункт "Свойства" в окне поиска, проводнике, и других окнах при клике по изображению.
При отображении свойств одной фотографии, находящейся в базе данных:

Отображается предпросмотр фотографии
Отображается комментарий к фотографии (можно изменять (если комментария у фотографии не было, то для отображения редактора комментария необходимо сделать двойной клик по предпросмотру))
Отображается имя файла и путь к нему
Отображается владелец и фотоальбом (можно изменять)
Отображается оценка (можно изменять)
Отображается дата фотографии (если она присутствует) (можно изменять)
Отображаются группы, к которым относится данная фотография. (можно изменять)
Размер фотографии
Ширина и высота
ID фотографии
Ключевые слова (можно изменять)
Включение в базовый поиск (можно изменять)
Ссылки (можно изменять) (см. ниже)

При отображении свойств нескольких фотографий, находящейся в базе данных:

Отображается комментарий к фотографии если он одинаков для всех фотографий (можно изменять для всех фотографий сразу)
Отображается папка, где размещаются фотографии (если они находятся в одной папке)
Отображается оценка, наиболее часто встречающаяся в этой группе (можно изменять оценку сразу для всех фотографий)
Отображается дата фотографий (если она присутствует и одинакова для всех) (можно изменять сразу для всех фотографий)
Отображаются группы, к которым относятся все эти фотографии.(можно изменять сразу для всех фотографий)
Общий размер всех фотографий
Ширина и высота (если она одинакова для всех фотографий)
Количество выбранных элементов
Ключевые слова (можно изменять сразу для всех фотографий, причём отображаются только общие для всех фотографий ключевые слова, а при добавлении или удалении из списка ключевых слов, они будут добавлены или удалены для каждой фотографии в отдельности)
Включение в базовый поиск (можно изменять)
Ссылки (можно изменять) (см. ниже)

После изменения свойств необходимо нажать на кнопку "Сохранить" для сохранения сделанных изменений.

При отображении свойств фотографии, отсутствующей в базе данных

Отображается предпросмотр фотографии
Отображается имя файла и путь к нему
Размер фотографии
Ширина и высота
Для добавления фотографии в базу данных необходимо нажать на кнопку "Добавить Файл"

Изменение ссылок к фотографиям.

Ссылки доступны нескольких типов:

ID - ссылка на определённы ID в БД. Удобно если файл может изменятся, но при экспорте БД может указывать на другой файл.
IDEx - ссылка на определённы IDEx фотографии, привязывается к изображению. Удобно если файл может перемещаться, сохраняется при экспорте. При изменении файла, на который ссылается данная ссылка нужно обновить эту запись в БД (из контекстного меню), или же если в момент изменения фотографии папка с файлом была открыта в проводнике. то программа автоматически обновит ссылку данного типа
Изображение - ссылка на файл, который будет открываться в окне просмотре (может не находится в БД)
Файл - ссылка на файл, по нажатию на ссылку будет выполнен запуск данного файла (файл любого типа)
Папка - ссылка на папку, при клике будет открыта в окне проводника
Текст - Позволяет добавить различного рада комментарии к фотографии или другие текстовые данные. (в тексте не может содержаться символ ";".

Для добавления ссылки выберите в меню "Добавить ссылку" и значение можно установить вручную, либо же нажав на кнопку "..." и выбрав нужный файл (значение будет интерпретировано исходя из типа ссылки). Также удобно "перетащить" нужный объект в окно редактирования ссылки.
