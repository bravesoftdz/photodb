Добавление фотографий
 

     Для добавления фотографий в базу данных необходимо:

В окне поиска  в контекстном меню над панелью задания условий поиска (обычной, не расширенной) выбрать пункт Панели->Проводник. В открывшемся окне проводника в контекстном меню для папки выбрать пункт "Добавить папку"
В окне просмотра фотографий в контекстном меню выбрать пункт (при просмотре фотографии, отсутствующей в Базе Данных) Добавить в БД -> Только этот файл\Все файлы в папке.
В окне проводника в контекстном меню выделенных фалов выбрать пункт "Добавить файл", или же нажатием по ссылке "Добавить объект(ы)" ("Добавить директорию)") в панели свойств.
В окне просмотра свойств при просмотре фотографии, отсутствующей в Базе Данных нажатием на кнопку "Добавить файл".

     Окно обновления служит для индикации процесса добавления фотографий и управления им. Это окно содержит следующие кнопки:

Прервать - Остановить добавление и удалить все элементы из очереди
Пауза\Запустить - Остановить\Запустить остановленный процесс добавления фотографий
Закрыть - Закрыть окно (добавление будет продолжаться в фоновом режиме)

    Контекстное меню:

Оставаться поверх всех окон - Окно будет отображаться поверх всех окон
Прозрачность - Задаёт прозрачность окна
Спрятать - Закрыть окно (добавление будет продолжаться в фоновом режиме)
Автоматически - Добавление фотографий начнётся автоматически после добавления фотографий в очередь добавления
Автоответ - Автоматический ответ при добавлении фотографий на диалог замены фотографий
  Нет - Не добавлять данные фотографии
  Заменять все - Заменить существующие записи на новые
  Добавлять все - Добавить все фотографии
  Пропустить все - Пропускает все фотографии
 История - хранит список объектов, которые не удалось добавить в БД (испорченные файлы или же зашифрованные, к которым не найдён пароль.
 Использовать сканирование по имени - пытается найти в БД фотографии с таким же именем файла и среди них "узнать" фотографию, при этом узнаётся фотография даже повёрнутая в другом редакторе

Если производится попытка добавить в БД фотографию, защищённую паролем, то она добавляется только при условии, что пароль будет сохранён в текущей сессии или настройках, т.е. фотографию будет возможно открыть, при этом запись в БД также автоматически зашифровывается.

При добавлении фотографий не рекомендуется выполнять другие действия с программой
