Проводник
 
     В базу данных встроен проводник, внешне схожий со стандартным проводником в Windows. Данный проводник позволяет выполнять все стандартные операции с файлами, как то удаление, переименование, копирование\вырезание\вставку файлов и папок, поддерживает сетевое окружение и виртуальную папку "Мой Компьютер". Поддерживает технология Drag&Drop (перетащи и отпусти), быстрые клавиши, и т.д.

     Горячие клавиши:

ctrl + X - Вырезать объект
ctrl + C - Скопировать объект
ctrl + V - Вставить объект из буфера обмена в текущую или выделенную директорию
ctrl +  -> - Вперёд
ctrl + <- - Назад (вернуться к предыдущей папке)
delele - Удалить объект в корзину
shift + delete - Удалить объект с диска
F2 - Переименование файла
F5 - Обновление папки

     Меню в проводнике:

Открыть в новом окне - Создать новое окно проводника и открыть там текущую папку
Скопировать - Скопировать текущую папку
Вырезать - Вырезать текущую папку
Вставить - Вставить из буфера обмена с текущую папку
Добавить папку - Добавить все фотографии, находящиеся в папке в базу данных
Создать новую папку - Добавить новую папку
Показать окно обновления - Открыть окно обновления
Открыть в окне поиска - Открыть окно поиска и показать там все файлы из Базы Данных, которые относятся к данной папке
Редактор - открытие  графического редактора
Печать - открытие диалога вывода изображений на принтер
Обновить - Перезагрузить папку
Изменить размер - Позволяет изменить размер и формат изображения (доступно при наличии пароля для этой фотографии)
Повернуть изображение - Позволяет повернуть изображение (доступно при наличии пароля для этой фотографии)
Конвертировать - преобразует изображение в другой формат (доступно при наличии пароля для этой фотографии)
Сделать рисунком рабочего стола - Устанавливает данное изображение как фон на рабочий стол, доступно для незашифрованных фотографий. (ВНИМАНИЕ: Microsoft Windows поддерживает далеко не все типы файлов, поэтому возможно перед установкой на рабочий стол файлов нестандартных форматов, их необходимо будет конвертировать)
Обновить запись в БД - Обновляет информацию о фотографии в БД. Необходимо, если фотография была изменена\отредактирована, а в БД хранится старый образ фотографии.
Экспорт изображений - позволяет экспортировать изображения с добавление личной подписи, изменения размеров фотографий и их поворотом.
Зашифровать файл - Зашифровывает файл.
Расшифровать файл - Расшифровывает файл.
Ввести пароль - Открывает фотографию и добавляет пароль для текущей сессии.
Выделить всё - Выделить все элементы
Перейти в окно поиска - Открыть окно поиска
Выход - Закрыть текущее окно

     Пункты в панели предпросмотра и задач:

Слайд-Шоу - Открыть данный файл во встроенном просмотрщике или же в другой программе просмотра, указанной в настройках.
Открыть - Открыть фотографию в приложение, которое предназначено для открытия файлов этого типа
Скопировать в - Скопировать выделенные объекты в указанную папку
Переместить в - Переместить выделенные объекты в указанную папку
Переименовать - Переместить выделенный объект
Свойства - Показать свойства выделенного объекта
Удалить - Удалить выделенный объект
Добавить объект\Добавить объекты - Добавить информацию о выделенных объектах в базу данных (если среди объектов папки, то добавляются и все подпапки)
Обновить - Обновить информацию о текущем объекте

     Пункты в главном меню программы:

Файл
Выход - Закрыть текущее окно

Редактировать
Назад - Назад (вернуться к предыдущей папке)
Вперёд - Вперёд (перейти к следующей папке)
Вверх - На один уровень вверх
Скопировать - Скопировать выделенные объекты (или же текущую папку при отсутствии выделенных объектов)
Вырезать - Вырезать выделенные объекты
Вставить - Вставить объекты, находящиеся в буфере обмена в текущую папку или же в выделенную папку
Выделить всё - Выделить все текущие объекты

Вид
Показать панель проводника - Отображать панель проводника для навигации по папкам
Показать панель с информацией - Отображать панель с информацией о выделенных объектах или текущей папке
Показывать личные - Отображать фотографии, помеченные в базе данных как личные
Показывать только общие - Не отображать фотографии, помеченные в базе данных как личные

Модули
Показать окно обновления - Открыть окно показа процесса обновления БД (добавления новых фотографий)
Управление БД - Открыть окно "Управление БД"
Поиск - При необходимости создать и перейти в окно поиска
Настройки - Отобразить окно настроек программы

Справка
Справка - Справка по программе
Активация программы - Ввод кода для активации программы
О программе - Окно "О программе"
Сайт программы - Открыть страничку программы
Связаться с автором - Написать письмо автору программы

        При установленной опции показа EXIF заголовков в файлах, фотографии с EXIF заголовками будут помечаться иконкой

Правой кнопкой мыши также можно вызвать контекстное меню из проводника Windows к этой фотографии, нажав правой кнопкой мыши и отпустив её более чем через секунду.
