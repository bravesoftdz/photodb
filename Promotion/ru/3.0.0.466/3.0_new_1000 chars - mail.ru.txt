Значительное обновление Photo Database 3.0

В новой программе добавлена возможность менять темы, доступно более 20 различных тем, часть из них поставляется с программой, остальные темы доступны для скачивания с сайта программы. 
Дополнительно добавлена возможность загружать фото любых поддерживаемых форматов и видео на Picasa Web непосредственно из программы. Программа автоматически сконвертирует и уменьшит размер фото при необходимости.
Многие современные фотоаппараты имеют встроенный GPS модуль который позволяет сохранить месторасположение фото в EXIF. Теперь с помощью Photo Database Вы можете отобразить эти фотографии на карте и посмотреть, где они были сделаны. Вы можете установить географическую метку для фото и самостоятельно привязать фото к местности.
Добавлена виртуальная папка для отложенных фото куда можно временно поместить понравившиеся файлы.
Для редактирования свойств фото теперь не обязательно добавлять их в коллекцию - окно свойств при сохранении автоматически добавит фото в коллекцию.
